# Mio Lagerstatus

Next.js-prototyp för att testa lagerstatuslogik för butik och leverans.

## Innehåll

- `src/components/lagerstatus-simulator.tsx` innehåller UI:t för kombinatorn.
- `src/lib/lagerstatus.ts` innehåller den typade logiken för scenarierna.
- `docs/Ny logik för lagerstatusar.md` är den funktionella specen.
- `docs/prototypes/lagerstatus-kombinator.html` är originalprototypen som portats.

## Kom igång

```bash
npm install
npm run dev
```

Öppna `http://localhost:3000`.
